form-1 (form요소들)

input의 설명문을 label태그로 나타내고 label에는 for의 값을, input에는 id의 값을 서로 일치시켜 줌

1. input type="text" 주관식. 한 줄 글상자

상자의 길이는 size로, 입력값의 최대길이 제한은 maxlength로

2. input type="password" 주관식. 암호 글상자

3. input type="checkbox" 다중 선택형 객관식. 체크박스

초기값으로 지정할 항목은 checked(checked="checked")를 추가함.

복습이 필요한 과목은?

4. input type="radio" 단일 선택형 객관식. 라디오버튼

단일 선택이 되려면 name의 값을 서로 일치 시킴.
초기값으로 지정할 항목은 checked(checked="checked")를 추가함.

가장 자신 있는 과목은?

5. textarea ... /textarea 주관식. 큰 글상자

가로 크기는 cols="n"로 세로 크기는 rows="n"로 지정

6. select ... /select 공간절약형 객관식. 펼침목록(드롭다운리스트)

등교에 걸리는 시간은?


- 초기값으로 지정할 항목은 selected(selected="selected")를 추가함.
등교에 걸리는 시간은?

- 다중 선택형 박스가 되려면 select에 multiple(multiple="multiple")를 추가함.
상담을 원하는 시간은?

- 형태만 박스가 되려면 select에 size="n"를 추가함.
상담을 원하는 시간은?

- 서로 관련있는 option들을 묶어서 그룹화하여 나타낼 수 있음.
option들을 optgroup태그로 묶고 optgroup에 label속성을 추가하여 해당 optgroup의 제목(주제)를 나타냄
상담을 원하는 시간은?

7. input type="hidden" 보이지 않는 정보 전송용

8. input type="file" 파일 전송용

9. input type="submit/reset/button" 전송/초기화/(일반) 버튼

value로 버튼의 글자를 지정



10. input type="image" 전송용 이미지

src로 이미지경로를 지정

11. button ... /button 전송용 버튼

초기화 하는 버튼은 type="reset"을 추가하고,
일반(폼과 무관한) 버튼은 type="button"을 추가
