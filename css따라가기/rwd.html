Responsive Web Design

Viewport

뷰포트는 사용자가 웹 페이지에서 볼 수 있는 영역입니다.
뷰포트 설정
브라우저가 페이지를 처음 로드할 때의 초기 확대/축소 레벨을 설정
<head>
<meta name="viewport" content="width=device-width, initial-scale=1.0"/>
</head>

반응형 이미지

max-width:n% 필요한 경우 축소되지만 원래 크기보다 더 크게 확대되지는 않습니다.

다양한 장치에 대한 다양한 이미지
<picture>
<source srcset="A-s.jpg" media="(max-width:npx)">
<source srcset="A.jpg">
<img src="A.jpg" alt="Flowers">
</picture>

Media Queries 미디어 쿼리

@media특정 조건이 참인 경우에만 CSS 속성을 사용
Breakpoint 중단점 추가
(ex)tablet 768px, desktop 1200px

미디어 쿼리 구문

@media mediatype and (min-width:n) {
선택자 {속성:속성값;}
}

미디어 쿼리 장치 종류 조건(mediatype)

  • all = Used for all media type devices
  • print = Used for print preview mode
  • screen = Used for computer screens, tablets, smart-phones etc.

미디어 쿼리의 조건

  • width and height of the viewport 뷰포트의 너비와 높이
  • orientation of the viewport (landscape/portrait) 뷰포트 방향(가로 또는 세로)
  • resolution 해상도

Always Design for Mobile First

모바일 퍼스트란 데스크톱이나 다른 기기를 디자인하기 전에 모바일을 먼저 디자인하는 것을 의미합니다.
(이렇게 하면 작은 기기에서 페이지가 더 빨리 표시됩니다.)
