display - 요소가 화면에 나타나는 방식 지정

  • display:inline; 한 줄에 나란히 옴.
  • display:block; 크기(width/height)와 margin 적용 됨.
  • display:inline-block; 한 줄에 나란히와 크기(width/height)와 margin 적용 됨.
  • display:none; 요소를 화면에서 숨김.

1. display:inline;

block tag - div

block tag - p

block tag - aside

2. display:block;

inline tag - a inline tag - b inline tag - span

3. display:inline-block;

block tag - div

block tag - p

block tag - aside
inline tag - a inline tag - b inline tag - span

4. display:none; (영역과 함께) 요소를 화면에서 숨김

block tag - div

block tag - p

block tag - aside
inline tag - a inline tag - b inline tag - span

5. visibility:hidden; (영역은 남기고) 요소를 화면에서 숨김

block tag - div

block tag - p

block tag - aside
inline tag - a inline tag - b inline tag - span
