transition 단순한 애니 효과

  • transition-property: ; 적용할 속성
  • transition-duration:ns/nms; 필수. 진행 시간. 단위는 초 또는 밀리초
  • transition-delay:ns/nms; 지연 시간. 단위는 초 또는 밀리초
  • transition-timing-function: ; 가속 감속 등 지정.
    • ease (기본값) 느리게 시작해서 빠르게, 그리고 느리게 끝
    • ease-in 느린 시작으로 지정
    • ease-out 느리게 끝나게 지정
    • ease-in-out 느리게 시작과 느린 끝으로 지정
    • linear- 시작부터 끝까지 동일한 속도
    • cubic-bezier(n,n,n,n) 3차 베지어 함수에서 사용자 고유의 값을 정의
      https://cubic-bezier.com/ ( bezier곡선값 자동생성사이트 )
