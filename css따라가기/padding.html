padding - 요소의 내부 여백

padding

1. padding의 방향별 적용

  • padding-top: n ; 위에 n만큼 적용
  • padding-right: n ; 우측에 n만큼 적용
  • padding-bottom: n ; 아래에 n만큼 적용
  • padding-left: n ;
    좌측에 n만큼 적용

2. padding의 값 갯수별 적용법

  • padding: n ; 양사방 n으로 동일
  • padding: n n ; 상하, 좌우가 서로 동일
  • padding: n n n ; 상, 좌우, 하 순으로 적용
  • padding: n n n n ; 상, 우, 하, 좌 순으로 적용
