position 요소의 위치 기준 지정

  • position:static; 기본값. 위치(거리) 적용 불가
  • position:relative; 현재 위치에서 거리 적용
  • position:absolute; 상위요소의 ~에서 거리 적용
  • position:fixed; 브라우저를 기준으로 ~에 고정
  • position:sticky; relative와 fixed의 믹스
  • left:n; 왼쪽에서 n거리 적용
  • right:n; 오른쪽에서 n거리 적용
  • top:n; 위에서 n거리 적용
  • bottom:n; 아래에서 n거리 적용

5. position:sticky; relative와 fixed의 믹스 브라우저의 특정 위치에 도달하면 고정 됨

1. position:static; 기본값. 위치(거리) 적용 불가

position:static; left: 1000px;

2. position:relative; 현재 위치에서 거리 적용

position:relative;
left:1000px;

position:relative;
right: 1000px;

position:relative;
top: 5em;

position:relative;
bottom: 5em;

position:relative;
left: 1400px; bottom: 400px;

position:relative;
left: 1200px; top: -400px;

3. position:absolute; 상위요소의 ~에서 거리 적용 ( 기준이 될 상위요소의 설정값 = position:relative )

position:absolute;
left: 1000px;

position:absolute;
right: 1000px;

position:absolute;
top:200px;
