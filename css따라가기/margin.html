margin - 요소의 외부 여백

margin

1. margin의 방향별 적용

  • margin-top: n ; 위에 n만큼 적용
  • margin-right: n ; 우측에 n만큼 적용
  • margin-bottom: n ; 아래에 n만큼 적용
  • margin-left: n ;
    좌측에 n만큼 적용

2. margin의 값 갯수별 적용법

  • margin: n ; 양사방 n으로 동일
  • margin: n n ; 상하, 좌우가 서로 동일
  • margin: n n n ; 상, 좌우, 하 순으로 적용
  • margin: n n n n ; 상, 우, 하, 좌 순으로 적용

margin을 양사방 적용시
block요소(줄이 바뀌는 태그) - 위아래로 인접한 요소들끼리는 서로 더하기가 아니라 큰 값이 적용 됨.
inline요소(줄이 안바뀌는 태그) - 위아래 margin은 적용 안되고, 좌우로 인접한 요소들끼리는 서로 더하기가 됨.
